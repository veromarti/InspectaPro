## Scope Description
This use case diagram represents the high-level interactions between users and the InspectaPro system.
It focuses on core business processes related to inspection management and execution.

## Actors Description

| Actor                 | Description                                                     |
| --------------------- | --------------------------------------------------------------- |
| Company Admin         | Manages inspection definitions, assignments and reviews results |
| Technician            | Executes assigned inspections and submits results               |
